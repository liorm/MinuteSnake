# MinuteSnake

Modern implementation of the classic Snake game using JavaScript and Canvas.

## Live Demo
Play the game here: [MinuteSnake](https://liorm.github.io/MinuteSnake/)

## Build Instructions
1. Install dependencies:
   ```bash
   npm install
   ```
2. Build the project:
   ```bash
   npm run build
   ```

## Test Instructions
Run the test suite:
```bash
npm run test
```

## Serve Locally
Start a local development server:
```bash
npm run serve
```
